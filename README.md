# SRI Platform - Backend Lambdas

AWS Lambda functions for Soluciones Rurales Integradas.

## Lambdas
- sri-auth-handler — autenticación JWT
- sri-token-authorizer — validación de tokens
- sri-settlements-list — listar liquidaciones
- sri-settlements-detail — detalle de liquidación
- sri-settlements-upload — subir PDFs a S3
- sri-settlements-parse — parsear PDFs
- sri-settlements-presign — generar URLs temporales
- sri-documents-handler — documentos
- sri-providers-handler — proveedores (datos estáticos)
- sri-stocks-handler — stocks
